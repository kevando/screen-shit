SCREEN SHIT

 This application helps you deal with screen shots. You can configure:


   *   Disable Mojave Screen Utility for faster action

   *   Copy screen shot image to clipboard without clicking a thing.

   *   Change directory that saves screen shot files. (Good bye clutter!)

   *  


       CMD + SHIFT + 3 takes a full screen screen shots

       CMD + SHIFT + 4 takes a partial screen screen shots
